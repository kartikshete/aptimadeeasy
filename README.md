# 🧠 AptiMadeEasy

 ![React](https://img.shields.io/badge/-React-blue?style=flat-square) ![Node.js](https://img.shields.io/badge/-Node.js-blue?style=flat-square) ![Productivity](https://img.shields.io/badge/-Productivity-blue?style=flat-square) ![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)

A dynamic educational platform focused on simplifying aptitude preparation using interactive modules and seamless navigation.

## 📝 Overview
AptiMadeEasy provides students and job seekers with a beautifully structured portal for breaking down complex quantitative, logical, and verbal reasoning topics. Built on Vite and Tailwind CSS v4, this platform focuses intensely on UI smoothness so learners can focus entirely on the material, not the interface.

## ✨ Key Features
- **High-Performance Architecture**: Powered by React 19 and Vite for instantaneous hot-reloads and lightning-fast client-side routing.
- **Axios Integrations**: Setup for rapid backend connectivity and dynamic question fetching.
- **Tailwind CSS v4 Integration**: Uses the latest Tailwind CSS Vite plugin for ultra-optimized styling and zero-bloat CSS.
- **Engaging UI**: Polished layouts utilizing `framer-motion` and `lucide-react`.

## 🛠 Tech Stack
- Frontend: `React 19`
- Styling Engine: `Tailwind CSS 4` + `PostCSS`
- HTTP Client: `Axios`
- Icons & Animation: `Lucide React` & `Framer Motion`

## 🚀 Getting Started

```bash
# Navigate to project
cd aptimadeeasy

# Install dependencies rapidly
npm install

# Fire up the blazing fast dev server
npm run dev
```


## 🌐 Deployment

### Vercel / Netlify (Recommended)
1. Push this repository to GitHub.
2. Connect your GitHub account to [Vercel](https://vercel.com) or [Netlify](https://www.netlify.com).
3. Select this project and use the following settings:
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`

### GitHub Pages
1. Install the gh-pages package: `npm install gh-pages --save-dev`
2. Add deployment scripts to your `package.json`.
3. Run `npm run deploy`.
